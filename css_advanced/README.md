# Get Schooled - Advanced CSS Project

> A fully responsive, accessible landing page built using only HTML and CSS — no JavaScript or frameworks.

![Screenshot](./screenshot.png)

## 📚 Project Overview

This project is part of the ALX "CSS, advanced" module. It demonstrates mastery of modern CSS layout techniques including Flexbox, Grid, and advanced selectors. The goal is to recreate a multi-section webpage based on a provided design using semantic HTML and clean, maintainable CSS.

---

## 🧠 Learning Objectives

After completing this project, you should be able to:

- Explain what a CSS rule is
- Differentiate between `id` and `class` selectors
- Center elements using modern CSS techniques
- Describe the Box Model and how it affects layout
- Understand CSS specificity and how it impacts styles
- Identify the difference between inline and block elements
- Use pseudo-classes effectively
- Compare Flexbox and CSS Grid layouts

---

## 🚀 Features

- 🎯 **Hero section** with a call to action
- 👥 **Testimonials** with avatars and bios
- ⭐ **Program highlights** showcasing key offerings
- 📊 **Statistics** section with clear, bold numbers
- ❓ **FAQ section** using clean and readable layout
- 🔚 **Footer** with relevant links and branding
